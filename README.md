# RetentionAgent: An AI-Driven Multi-Agent Framework for Employee Retention Analysis

## 📌 Overview
**RetentionAgent** is an AI-driven employee retention analysis framework that integrates machine learning, natural language processing, economic modeling, and multi-agent reasoning into a single, cohesive analytical system.
